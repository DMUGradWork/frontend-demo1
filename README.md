# GradWork Frontend
GradWork 프로젝트의 모바일 앱 프론트엔드입니다.<br>
이 앱은 **React Native**와 **Expo**를 사용하여 개발되었습니다.


## 실행 시 설치 요구 사항

- Node.js
- yarn
- Expo CLI
- Xcode or Android Studio

## 기술 스택

- **React Native**: 크로스 플랫폼 모바일 앱 개발을 위한 프레임워크
- **Expo**: 앱을 쉽게 개발하고 배포할 수 있게 돕는 도구
- **JavaScript**
